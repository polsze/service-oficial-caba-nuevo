Service Oficial CABA

Landing page desarrollada para Service Oficial CABA, empresa especializada en reparación de heladeras a domicilio en la Ciudad Autónoma de Buenos Aires (CABA).

El proyecto fue diseñado con un enfoque orientado a la conversión de clientes, priorizando la experiencia de usuario (UX/UI), la velocidad de carga y el posicionamiento orgánico en buscadores (SEO).

Características
Diseño responsive para dispositivos móviles, tablets y escritorio.
Optimización SEO on-page para búsquedas relacionadas con reparación de heladeras en Capital Federal.
Formularios de contacto para generación de consultas.
Botones de llamada y WhatsApp con acceso rápido.
Secciones informativas orientadas a generar confianza y aumentar conversiones.
Optimización de rendimiento y carga de imágenes.
Tecnologías
React
Vite
Tailwind CSS
Lucide React
Objetivo

Crear una presencia digital moderna, rápida y optimizada para captar potenciales clientes que buscan servicio técnico de heladeras a domicilio en CABA.
